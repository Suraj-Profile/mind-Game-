MathMind Evolution — Run Instructions

Simple instructions to run the app locally.

Options:

- Open directly: double-click `index.html` in the project root.

- Recommended (local server): run the included `start-server.bat` (Windows).

  - If Python 3 is installed, the script will start a local server on port 8000 and open your browser at `http://localhost:8000`.
  - If Python is not available the script will just open `index.html` directly.

Troubleshooting:

- If the UI shows unexpected values, open the browser devtools console (F12) and look for errors.
- Ensure your browser allows localStorage access (not blocked by privacy settings).

Enjoy!
